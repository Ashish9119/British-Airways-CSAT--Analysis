# ✈️ British Airways Customer Satisfaction (CSAT) Analysis

## 📌 Project Overview

This project analyzes British Airways customer reviews using **Power BI** to understand customer satisfaction, service quality, travel experience, and geographical trends.

The dashboard helps identify the factors affecting customer satisfaction and provides actionable business insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze overall customer satisfaction.
- Measure recommendation percentage.
- Compare ratings across traveller types and seat classes.
- Identify geographical trends.
- Evaluate service quality.
- Build an interactive multi-page Power BI dashboard.

---

## 🛠 Tools Used

- Power BI Desktop
- Microsoft Excel
- DAX
- Data Modeling
- Power Query

---

## 📂 Dataset

The dataset contains customer reviews with information such as:

## 📊 Key Insights

- Average customer rating is **4.19/5**.
- Around **42%** of customers recommend British Airways.
- Solo Leisure travellers provide the highest ratings.
- Business Class and First Class generally receive better ratings.
- Europe contributes the highest number of customer reviews.
- Entertainment and Food & Beverages receive comparatively lower ratings.
- Customer reviews declined after 2019.
- Service quality differs significantly across traveller and seat categories.
